# zym user guide

zym is a keyboard-driven code editor for Linux, built on GTK 4. It combines
modal (vim) editing, Spacemacs-style `space`-leader commands, LSP code
intelligence, tree-sitter highlighting, deep git/GitHub integration, and
first-class support for running coding agents alongside your work.

- [Getting started](getting-started.md) — install, first launch, core concepts
- [Keybindings](keybindings.md) — the leader scheme, full reference, customization
- [Vim keymap reference](vim-keymap-reference.md) — every vim binding, per mode
- [Configuration](configuration.md) — settings, the preferences window
- [Git & GitHub](git.md) — staging, diffs, history, branches, pull requests
- [Agents](agents.md) — running coding agents inside the workbench
- [Troubleshooting](troubleshooting.md) — install and runtime issues

Press `space ?` inside zym at any time to see every active keybinding and where
it comes from.
